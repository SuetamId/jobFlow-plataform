# JobFlow

JobFlow is a modern job marketplace frontend built with React, designed as a portfolio project to demonstrate strong software architecture, clean code practices and scalable design.

The platform simulates a real-world job portal experience, supporting both candidates and recruiters, with all data mocked and no backend dependency.

## Key Features

- Public landing page with job listings and search
- Mocked authentication (login and register)
- Candidate area:
  - Job search and filtering
  - Apply to jobs
  - Save jobs
  - Dashboard overview
- Recruiter area:
  - Company job management
  - Job applications review
- Fully responsive and accessible UI
- SEO-friendly and high performance

## Technical Highlights

- React (SPA)
- Domain-Driven Design (DDD)
- Clean Architecture
- Clear separation of concerns
- Mocked infrastructure ready to be replaced by real APIs
- Focus on maintainability and scalability

## Purpose

This project was created for learning and portfolio purposes, focusing on architecture and code quality rather than backend integration.
